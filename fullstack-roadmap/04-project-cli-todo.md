# 项目 1：命令行 Todo

## 项目定位

这是第一个项目，不追求复杂功能。目标是把 Go 基础语法变成项目手感。

你要练的是：

- package 拆分。
- struct 设计。
- error 处理。
- JSON 持久化。
- 单元测试。

## 功能范围

- [x] 新增任务。
- [x] 完成任务。
- [x] 删除任务。
- [x] 列出所有任务。
- [x] 按状态筛选任务。
- [ ] 使用 JSON 文件保存数据。
