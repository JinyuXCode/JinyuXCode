## Roadmap

这套路线面向“前端转 Go 全栈”。它不是知识点堆叠，而是按学习曲线拆成里程碑，围绕一个持续演进的项目推进。

```text
Read AI-written code
  -> Gin service layering
  -> Database modeling
  -> Auth and permission
  -> RBAC and admin
  -> Cache / rate limit
  -> Docker / deployment
  -> Business system project
```

学习计划与项目代码在独立的私有仓库中维护。

## Operating Notes

```text
I do not want to only "use AI to write code".

I want to:
- describe the problem clearly
- split the work into stable modules
- review generated code critically
- test the behavior locally
- understand the tradeoffs
- ship the feature
```
